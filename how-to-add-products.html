SECTION 01
HOW CREATOR TRACKING WORKS
Every creator gets their own unique link to your website. When someone visits through that link, every Buy button on the site automatically includes that creator's tracking ID.
CreatorTheir link to shareWhat ClickBank records
Johnyoursite.github.io?creator=johntid=john
Sarayoursite.github.io?creator=saratid=sara
Mikeyoursite.github.io?creator=miketid=mike
No creatoryoursite.github.iotid=default
How it works
The website reads ?creator=john from the URL automatically. It then adds ?tid=john to every single Buy button on the page. You do nothing — it happens instantly when the page loads.
SECTION 02
HOW TO ADD A NEW PRODUCT
Open your index.html file in any text editor (Notepad on Windows, TextEdit on Mac). Find the PRODUCTS section and copy-paste a new product block.
1
Open index.html in a text editor
Right click the file → Open with → Notepad (Windows) or TextEdit (Mac)
2
Press CTRL+F (or CMD+F on Mac) and search for:
const PRODUCTS = {
3
Copy this template and paste it inside the PRODUCTS block
Add a comma after the previous product, then paste your new one
// COPY THIS TEMPLATE FOR EVERY NEW PRODUCT:

newproduct1: {
  emoji: '🎯',                // any emoji
  niche: 'Anime',            // Anime / Gaming / Fitness
  title: 'Your Product Name',  // product title
  desc: 'Short description of the product goes here.',
  features: [
    'Benefit one',
    'Benefit two',
    'Benefit three',
    'Money-back guarantee'
  ],
  hasVideo: false,            // true or false
  cbPath: 'nick.PRODUCT.hop.clickbank.net' // your real ClickBank hoplink
},
Important
Replace nick.PRODUCT.hop.clickbank.net with your real ClickBank hoplink. You get this from ClickBank Marketplace → Promote → Generate Hoplink.
SECTION 03
HOW TO ADD THE PRODUCT CARD ON THE PAGE
After adding the product to PRODUCTS, you need to add the card to the page. Find the section for your niche and paste this card HTML.
<!-- COPY THIS CARD TEMPLATE -->

<div class="product-card">
  <div class="product-thumb">🎯</div>
  <div class="product-body">
    <div class="product-niche">Anime</div>
    <div class="product-title">Your Product Name</div>
    <div class="product-commission">$97 · 75% commission</div>
    <div class="product-btns">
      <button class="card-btn card-btn-info" onclick="openModal('newproduct1')">ℹ Info</button>
      <button class="card-btn card-btn-buy buy-btn" data-product="newproduct1">Buy Now</button>
    </div>
  </div>
</div>
Make sure
The ID in openModal('newproduct1') and data-product="newproduct1" must exactly match the key you used in the PRODUCTS block above.
SECTION 04
HOW TO GIVE CREATORS THEIR LINKS
Once your site is live on GitHub, just send each creator their personal link. That's it — no other setup needed.
Creator nameLink you send them
Johnyourusername.github.io?creator=john
Sarayourusername.github.io?creator=sara
Mikeyourusername.github.io?creator=mike
Any nameyourusername.github.io?creator=THEIRNAME
Pro tip
You can create unlimited creators just by changing the name after ?creator= — no code changes needed. Just make up a new link and give it to the new creator instantly.
SECTION 05
HOW TO CHECK SALES IN CLICKBANK
Log into your ClickBank account and go to Reporting → Sales. You will see every sale with the tid (creator name) attached.
1
Log into clickbank.com
Go to your account dashboard
2
Click Reporting → Sales
You will see a list of all sales this week
3
Look at the TID column
It shows john, sara, mike — exactly who made each sale
4
Count each creator's sales and pay them via PayPal
Every Friday count up each TID and send their cut
